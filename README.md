# AI Sales Predictor

AI Sales Predictor is a web-based machine learning demo that predicts sales trends from CSV data.

## Features
- Upload CSV dataset
- Automatic column detection
- Linear Regression prediction
- Interactive charts using Chart.js
- Forecast future sales with confidence range

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run
1. Download or clone the repository.
2. Open `index.html` in a browser.
3. Upload a CSV file with numeric sales data.
4. Configure columns and run the prediction.
